Epoya Game

Epoya is going to be (currently under heavy development) a top-down, 2D, vector art RPG game. The game is intended to be an open-world, sandbox, "horizontal progression" game inspired by games like Minecraft, Old School Rune Scape, and Stardew Valley.

The game will feature multiple races, crafting, co-op play, exploration, economy, fishing, hardcore mode, quests, secret finding, housing, and more. The "killer feature" of the game will be the main story and a high degree of player agency (inspired by Undertale) that will make each adventurer's journey unique.

At this point (mid-2026), the game is in very early development by a solo, indie, (inexperienced) game developer. It is likely to be quite some time before there is anything worth showing. Below is a mirror of GitHub activity to show that the game is actually being worked on slowly but surely. In the future, there will be devlogs on YouTube and a Steam page.
